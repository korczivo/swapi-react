# Game based on Star Wars API

## Requirements

Node v14.19.0 in your environment or nvm installed.

## How to run project

If you are using NVM you can use command below

```
nvm use
```

Install dependencies 
```
yarn install
```

Start project
```
yarn start
```

Now project is running here: http://localhost:3000/
